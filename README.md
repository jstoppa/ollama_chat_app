# Ollama Chat app

Very simple JavaScript Chat app to connect to locally hosted Ollama API.

![Alt text](sample-app.webp)

## Setup

1. Setup Ollama locally
2. Clone repo
3. Run npm install

```bash
npm install
```

4. Install Parcel

```
npm install -g parcel-bundler
```

4. Run app

```bash
parcel src/index.html
```
